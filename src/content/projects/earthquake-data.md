## 项目概述

《U3D 地震数据可视化》是我在昆明理工大学研究团队参与的数据可视化项目。我担任 Unity 开发者和研究助理，参与了从数据分析到交互展示的完整流程。

## 工作流程

- 使用 R 对地震数据进行分析
- 使用 ArcGIS 进行空间可视化处理
- 使用 Blender 构建进一步展示所需模型
- 最终通过 Unity 3D 与 ARKit 实现可交互的可视化体验

## 项目价值

这个项目让我把技术开发与研究型表达结合起来，也锻炼了我与多学科工作流协作的能力，不只是做“能跑起来”的内容，而是把复杂数据转化为可理解、可交互的体验。
