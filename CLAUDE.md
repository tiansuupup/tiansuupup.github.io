# Portfolio 项目上下文

## 项目概述
个人 Portfolio 网站，展示游戏开发作品和技能。

## 技术栈
- Astro 5 + React 19 + TypeScript
- Tailwind CSS 4
- Framer Motion

## 代码规范
- 使用 TypeScript strict 模式
- 组件使用 PascalCase 命名
- 优先使用 named exports
- CSS 优先用 Tailwind，复杂动画用 Framer Motion
- 图片使用 Astro 的 <Image> 组件优化

## 项目结构
src/
├── components/    # React 交互组件
├── layouts/       # 页面布局
├── pages/         # 路由页面
├── content/       # Markdown 内容（项目介绍等）
├── styles/        # 全局样式
└── assets/        # 图片、字体等静态资源

## 设计风格
赛博朋克 × 像素艺术的融合美学——致敬游戏开发背景。
深色主题为主，霓虹色点缀，微妙的 glitch 特效和流畅的过渡动画。
