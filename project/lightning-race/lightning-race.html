Lightning Race CarRacing Game

Lightning Race Car is a 3D racing game built in Unity HDRP. The game is about racing with other AI racer, win prizes, then buy or upgrade cars.

The track and the default car were built in Blender.

In the game there are cool speediness system, dynamic difficulty system, Nitro-boost system, checkpoint system and enemy AI navigation system.

I made this game because I am a fan of the Need For Speed series and the Forza Horizon series.
